# MakerRent
## Идея
Сервис аренды дорогого оборудования для хобби и мастерских между мастерами.
## Elevator Pitch
Многие мастера и инженеры-любители отказываются от своих идей, так как покупка условного 3D-принтера или лазерного гравера стоит десятки тысяч рублей. 
Для решение данной проблемы создается платформу для аренды оборудования между мастерами. В отличие от обычных досок объявлений, здесь обеспечивается верификация исправности техники и страхование каждой сделки. 
В результате владельцы получают доход от простаивающей техники, а мастера — доступ к оборудованию по цене нескольких чашек кофе.
## Lean Canvas
| Блок | Содержание |
| :--- | :--- |
| **Проблема** | Высокая стоимость оборудования; простой техники у владельцев; риски поломок при аренде. |
| **Сегменты потребителей** | Студенты, DIY-энтузиасты, инженеры. Ранние последователи: участники 3D-сообществ. |
| **Уникальная ценность** | Профессиональная мастерская в совокупности с гарантией безопасности. |
| **Решение** | Приложение с картой техники, верификация оборудования и система безопасных платежей. |
| **Каналы** | Профильные форумы, Telegram-сообщества технарей, реклама в DIY-магазинах. |
| **Потоки прибыли** | Комиссия со сделок (15%), платное продвижение объявлений, страховые взносы. |
| **Структура издержек** | Разработка приложения, маркетинг, поддержка пользователей, юридическое сопровождение. |
| **Ключевые метрики** | Количество активных лотов, объем сделок, индекс удовлетворенности пользователей. |
| **Скрытое преимущество** | Уникальная система страхования DIY-оборудования и база проверенных мастеров. |
